# **App Name**: Namo Vṛkṣebhyaḥ Donation Portal

## Core Features:

- Landing Page: Display key information about the Geet Sangeet Sagar Trust and its mission for tree plantation and adoption.
- E-Brochure Link: Provide a prominent link to view the trust's e-brochure.
- Tree Plantation and Adoption Form Link: Offer a direct link to the tree plantation and adoption form.
- Contact Information: Allow users to view contact information for the trust.
- About Section: Allow users to learn more about the trust and its activities.
- Responsive Design: Ensure the website is accessible and functional across various devices (desktops, tablets, and mobile phones).

## Style Guidelines:

- Primary color: Light green (#E8F5E9) to evoke nature and growth.
- Secondary color: Light blue (#E3F2FD) for a clean, airy feel.
- Font recommendation: 'PT Sans', a humanist sans-serif that combines a modern look and a little warmth or personality, suitable for headlines or body text
- Place the texts '॥ विजयते श्रीबालकृष्णः प्रभुः ॥' and '॥ श्रीवल्लभविट्ठलेशौ विजयेते ॥' in fontsize 8 above the heading.
- Center the main heading 'Namo Vṛkṣebhyaḥ Vṛkṣāropaṇa Mahotsava' with large font size on the landing page.
- Display the trust logo beneath the main heading.
- Arrange the 'View E-brochure' and 'Fill Tree Plantation and Adoption Form' buttons in a vertical, up-down format.
- Design the 'View E-brochure' and 'Fill Tree Plantation and Adoption Form' buttons with a bubble-like texture featuring whitish and light-blueish hues.
